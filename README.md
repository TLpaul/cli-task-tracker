# CLI Task Tracker

A lightweight command-line task tracker built in Python. Tasks are stored locally using a JSON file.

## Features
- Add tasks
- List tasks
- Mark tasks as completed
- Persistent storage via `tasks.json`
- No external dependencies

## Requirements
- Python 3.8+

## Quick Start

Clone the repo:
```bash
git clone https://github.com/<YOUR_USERNAME>/cli-task-tracker.git
cd cli-task-tracker
